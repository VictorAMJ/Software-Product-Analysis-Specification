# Projeto Hotel Sustentável

## Pré-requisitos

- Python 3.12+

## Como rodar

1. Clonar o projeto:  
    ```bash
    git clone <URL_DO_REPOSITORIO>
    cd Projeto
    ```

2. Instalar dependências:  
    ```bash
    pip install -r requirements.txt
    ```

3. Rodar a aplicação Flask:  
    ```bash
    python app.py
    ```

4. Acesse a aplicação em:  
    ```
    http://127.0.0.1:5000
    ```
